Logmyday Android App

An android application that simplifies keeping a log of daily values

View on GitHub Download .zip Download .tar.gz

This is app is a little side project I wrote in a few days to keep my personal log. It allows you to write down and store some daily values, whether it's your mood, your grades, or some medical information, anything you may want to keep track of. It also allows inserting notes (so that it can be used to keep a journal), and setting a daily reminder. The inserted data can be exported as csv.

Download

You can download the apk from here The only required permission is for writing to the sd card in order to export your data. This app is currently compatible with android 4.1 onwards.

About your data

As this is only a little side project, I take no responsibility for the integrity of what you write into the application. For what concerns privacy, however, everything you write stays in the private database of the application, and can only be read by other applications (or sent via internet) after it is exported using the apposite function.
